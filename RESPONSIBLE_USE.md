# Responsible Use Policy

## Purpose

TaskSchedulerPersist is a security research tool created for the following purposes:

- **Education** -- Understanding how Windows Task Scheduler persistence works at the COM interop level, enabling security professionals and students to study real-world techniques.
- **Authorized security testing** -- Supporting red team engagements, penetration tests, and adversary simulations conducted under explicit written authorization.
- **Capture The Flag (CTF) competitions** -- Providing a reference implementation for competitive security exercises in controlled environments.
- **Detection engineering** -- Enabling blue teams and detection engineers to develop, test, and validate detection rules and monitoring capabilities against known persistence techniques.
- **Academic research** -- Facilitating published or peer-reviewed security research in university and laboratory settings.

## Prohibited Uses

The following uses of this software are strictly prohibited:

- **Unauthorized access** -- Deploying this tool on any system without explicit written authorization from the system owner.
- **Malware development or distribution** -- Incorporating this code into malware, ransomware, spyware, or any software intended to cause harm.
- **Violation of laws or regulations** -- Using this tool in any manner that violates local, state, national, or international laws, including but not limited to the Computer Fraud and Abuse Act (CFAA), the General Data Protection Regulation (GDPR), and equivalent legislation in other jurisdictions.
- **Unauthorized surveillance** -- Using this tool to monitor, track, or surveil individuals without their knowledge and consent.
- **Commercial exploitation of unauthorized access** -- Using knowledge or access gained through this tool for financial gain through unauthorized means.
